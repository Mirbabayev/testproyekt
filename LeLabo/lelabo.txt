# LE LABO VEBSAYT DİZAYN ELEMENTLƏRİ

## ŞRİFTLƏR VƏ TİPOQRAFİYA
- Əsas şrift: Helvetica Neue və ya oxşar sans-serif şriftlər
- Menyu elementləri: BÖYÜK HƏRF, ultra-nazik şrift çəkisi (Ultra Light/Thin)
- Başlıqlar: Nazik şrift çəkisi (Light/Thin)
- Məhsul adları: BÖYÜK HƏRF, normal şrift çəkisi
- Mətn korpusu: Light/Regular şrift çəkisi, kiçik mətn ölçüsü
- Şrift hündürlüyü: Sıx sətir aralığı, minimalist görünüş
- Alt başlıqlar: Kiçik şrift ölçüsü, boz rəng
- Düymə mətnləri: Kiçik hərf, nazik şrift çəkisi, minimal kənar boşluqlar

## RƏNG PALETİ
- Ana rənglər:
  - Ağ (#FFFFFF) - dominant arxa fon rəngi
  - Qara (#000000) - əsas mətn rəngi
  - Açıq bej/krem (#F5F5F0) - ikinci dərəcəli arxa fon rəngi
  - Tünd boz (#333333) - ikinci dərəcəli mətn rəngi
  - Açıq boz (#CCCCCC) - ara xətlər və ikinci dərəcəli elementlər
- Minimal rəng istifadəsi
- Yüksək kontrast
- Kölgələr və gradient yoxdur, hamısı solid/tam rənglərdir
- Sadə, təbii və təmiz görünüş

## LAYOUT (TƏRTIBAT) SİSTEMİ
- 12 sütunlu grid sistemi
- Bol boş sahə istifadəsi
- Asimmetrik layout - sol tərəfdə daha çox məzmun
- Səliqəli düzüm və hizalamalar
- Məzmun blokları arasında böyük məsafələr
- Məzmunların hizalanması: sol-üst hizalama əsasən
- Horizontal xətlər ayırıcı kimi istifadə edilir
- Elementlər arasında böyük neqativ sahə

## UI (İSTİFADƏÇİ İNTERFEYSİ) KOMPONENTLƏRİ

- Açılan menyular:

- İkonlar:

# LE LABO VEBSAYT UX (USER EXPERIENCE) ELEMENTLƏRİ

# LE LABO DİZAYN VƏ UX İLHAM NOTLARI

## DİZAYN FƏLSƏFƏSİ
Le Labo vebsaytı minimalist, təmiz və praktik bir dizayn fəlsəfəsi daşıyır. Bu fəlsəfə aşağıdakı prinsiplərdən ibarətdir:

1. **Az, Lakin Daha Çox Prinsip**: Minimalist dizayn, yalnız lazımi elementlərin saxlanması
2. **Məzmun Əsaslı Dizayn**: Məhsulun özünə və məlumatlarına diqqəti çəkmək
3. **Laboratoriya Estetikası**: Elmi/laboratoriya görünüşü, texniki detallar
4. **Əl İşçiliyi Vurğusu**: Sənətkarlıq və şəxsi toxunuş hissi
5. **İnformasiya Açıqlığı**: Məhsullar və məlumatların aydın təqdimatı
6. **Lüks, Lakin Sadə**: Bahalı hiss, lakin həddən artıq dəbdəbəli olmayan

## DİZAYN İSTİFADƏSİ ÜÇÜN ƏSAS NÖQTƏLƏR

### 1. Şriftlər və Tipoqrafiya
- Sans-serif şriftlər istifadə edilməli (Helvetica Neue, Arial və ya oxşarları)
- Mətnlərin çoxu BÖYÜK HƏRF ilə yazılmalı, xüsusən də naviqasiya və başlıqlarda
- Nazik şrift çəkiləri (300 və ya light) əsas mətnlər üçün
- Şrift ölçüləri kiçik olmalı, məsələn:
  - Ana menyu: 12-14px
  - Başlıqlar: 18-22px
  - Mətn: 12-14px
- Sətir aralığı sıx olmalı
- Mətn blokları qısa və yığcam tutulmalı

### 2. Rəng Paleti
- Dominant ağ fon
- Qara mətn
- Açıq bej/krem aksent
- Açıq boz ikinci dərəcəli mətn
- Konkret rəng kodları:
  - Ağ: #FFFFFF
  - Qara: #000000
  - Bej: #F5F5F0
  - Açıq boz: #CCCCCC
  - Tünd boz: #333333

### 3. Layout və Boşluq İstifadəsi
- 12 sütunlu grid sistemi əsas alınmalı
- Bol boş sahə (negative space) istifadə edilməli
- Elementlər arasında böyük məsafələr
- Asimmetrik layout, sol tərəfə əyilmiş məzmun
- Məsafələr arasında harmoniya - eyni boşluq ölçüləri
- Responsive dizayn üçün breakpointlər:
  - Mobile: 360px - 767px
  - Tablet: 768px - 1023px
  - Desktop: 1024px+

### 4. UI Elementləri
- Düymələr minimal stilində, incə sərhədlər və ya sərhədsiz
- Formlar sadə, alt xətlər ilə
- İkonlar kiçik və incə
- Qısa və aydın etiketlər
- Hover effektləri sadə rəng dəyişmələri
- Məhsul kartları sadə və funksional, artıq bəzəklərdən uzaq

### 5. Şəkillər və Vizuallar
- Yüksək keyfiyyətli məhsul fotoları
- Ağ, təmiz arxa fon
- Neytral işıqlandırma
- Minimalist kompozisiya
- Məhsullar mərkəzdə, frontal görünüş
- Artıq filtr və effektlərdən qaçınmaq

## UX DİZAYN PRİNSİPLƏRİ

### 1. İstifadəçi Yolları (User Flows)
- Sadə və birbaşa naviqasiya
- Az klik ilə məhsullara çatma
- Məntiqi kateqoriya bölgüləri
- Aydın informasiya arxitekturası
- Asan axtarış imkanı

### 2. İnteraktiv Elementlər
- Sadə və intuitiv interfeys
- Minimal animasiyalar
- Dönüş bildirişləri (feedback)
- Fade-in/fade-out keçidlər
- Görünməz, lakin effektiv mikro-interaksiyalar

### 3. Məhsul Nümayişi
- Məhsul şəkli dominant
- Yığcam, lakin tam məlumat
- Ölçü və qiymət aydınlığı
- Xüsusiyyətlərin sadə təqdimatı
- Tərkib və istifadə haqqında dəqiq məlumat

### 4. Məhsul Fərdiləşdirməsi
- Sadə və istifadəçi dostu proses
- Canlı önbaxış
- Aydın məhdudiyyətlər
- Yaxşı izahat
- Çevik düzəliş imkanı

### 5. Səbət və Ödəniş
- Şəffaf qiymətləndirmə
- Minimum addımlı ödəniş prosesi
- Vizual dəstək (mərhələlər)
- Dönüş bildirişləri
- Səhv düzəltmə asanlığı

## İLHAM ALMAQ ÜÇÜN NÖVBƏTI ADDIMLAR

### 1. Dizayn Örnəyi Üçün
Əgər Le Labo estetikasının eyni və ya bənzər sahədə tətbiq edilməsini planlaşdırırsınızsa:
- Laboratuar/tibbi/elmi temalardan istifadə edin
- Nömrələr və kodlar ilə məhsul adlandırması
- Texniki spesifikasiya formatında məlumat
- Tibbi etiketlər kimi etiketlər dizayn edin

### 2. Digər Sahələrə Adaptasiya Üçün
Əgər bu estetikanı başqa bir sahə üçün adaptasiya etmək istəyirsinizsə:
- Əsas minimalist prinsipləri qoruyun
- Sahənizə uyğun terminologiya istifadə edin
- Laboratoriya stilini sahənizə uyğun transformasiya edin
- Struktur və grid sistemini saxlayın, məzmunu dəyişdirin

### 3. Mobil Dizayn Üçün
- Burger menyu ilə əsas naviqasiya
- Tək sütunda məhsul siyahısı
- Böyük şəkillər, kiçik mətn
- Touch-friendly elementlər
- Kompakt, lakin oxunaqlı məlumat

### 4. Brending Konsequentliyi
- Bütün səhifələrdə eyni şrift və rəng
- Tanınan şablon (template) 
- Naviqasiya təcrübəsində konsequentlik
- Məhsul təqdimatında vahid stil
- Brend xüsusiyyətlərinin hər yerdə vurğulanması

## XÜLASƏ: ƏSAS GÖTÜRÜLƏCƏK NÖQTƏLƏR

1. **Minimalizm**: Az element, çox boşluq, daha təsirli dizayn
2. **Tipoqrafiya Əhəmiyyəti**: BÖYÜK HƏRF, nazik şrift çəkisi, sıx sətir aralığı
3. **Laboratuar Estetikası**: Tibbi/elmi baxış, texniki məlumatlar
4. **Funksionallıq**: İstifadəçi təcrübəsini ön planda tutmaq
5. **Məhsul Fokuslanması**: Məhsulları və keyfiyyəti ön plana çıxarmaq
6. **Dizayn Harmoniyası**: Bütün elementlərin biri-biri ilə əlaqəli olması
7. **Şəxsiləşdirmə**: İstifadəçiyə özəl təcrübə təqdim etmək
8. **Konsequentlik**: Bütün platformalarda eyni stil və təcrübə

Le Labo vebsaytından ilham alaraq, sadə, funksional və estetik cəhətdən zəngin bir istifadəçi təcrübəsi yaratmaq mümkündür. Minimal dizayn, keyfiyyətli məzmun və məhsul fokuslu yanaşma ilə istənilən sahədə təsirli nəticələr əldə etmək olar.
